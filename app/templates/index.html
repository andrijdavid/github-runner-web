How to connect this runner to a repository
  1. Open your repository on GitHub and go to Settings → Actions → Runners.
  2. Click New self-hosted runner, choose Linux, and copy the token shown in the configure command.
  3. Paste the repository URL and token below, choose a name and labels, then click Attach runner.

You can also use an organization URL (for example https://github.com/myorg) to register an organization runner.
