How to connect this runner
  1. Open your repository on GitHub and go to Settings → Actions → Runners.
  2. Click New self-hosted runner, choose Linux, and copy the token shown in the configure command.
  3. Paste the URL and token below, set a name and labels, then click Attach runner.

You can also use an organisation URL (for example https://github.com/myorg) to register an organisation runner.
